프로젝트 소개

인기있는 영화 드라마 정보를 한번에!

Netflix   Watcha   Wavve

여가시간에 굳이 이것저것 추천영상을 찾아 보지 않아도 사이트에서 회사별 인기있는 영화나 드라마를 추천해 고민 없이 손쉽게 영상을 찾을 수 있다.

요즘같이 코로나로 인해 집에서 여가시간을 보내는 사람들은 넷플릭스 같은 사이트에서 영화나 드라마를 자주 보곤 합니다. 그럴 때 보통 영화 추천이라고 검색을 하면 블로거나 유튜버 한 명의 의견이 많이 들어가있기 때문에 본인과 맞지 않은 영화를 시청하는 경우가 다분합니다.
위 사이트는 넷플릭스 왓챠 웨이브의 인기 있는 영상을 모두 볼 수 있게 추천하고 많은 사람들의 리뷰로 보고 싶은 영화나 드라마를 빠르게 고를 수 있습니다.

체크리스트 · DB 테이블 상세표

체크리스트 작성을 통해 필요기능에 어떤 기능을 구현할지 생각해 보았고, 기능설명을 통해 각 기능별 세부기능들을 나눠서 계획을 짜보았습니다.
프로젝트를 진행하면서 리뷰등록과 자유게시판에 문제점이 생겨 다시 체크리스트를 확인하여 방향을 잡을 수 있었습니다.

DB테이블 상세표 회원 · 자유게시판 · 리뷰등록 순으로 각 테이블의 이름 유형을 알아보았습니다.
자유게시판과 리뷰의 번호에는 AUTO_INCREMENT값으로 1씩 증가시켰습니다.

개발 환경

  • Framework
    Spring Boot :
    일일이 모든 설정을 하지 않아도 자주 사용되는 기본설정을 알아서 해주어 많은 도움이 되었습니다.

  • language
    HTML · CSS · JavaSpript · MariaDB · Java :
    HTML로 전체적인 웹의 구조를 짜고 CSS로 정적인 디자인을 완성하고 JS로 동적인 기능들을 추가하였습니다.

  • Tool
    Intellij · VisualStudio :
    프로젝트 진행시 벡엔드,프론트엔드에 다양한 도움을 주었고, 자동완성, 소스코드분석등 이클립스와 비교했을때 너무 좋아서 잘사용하였습니다.

  • Library
    jQuery · Thymeleaf :
    Thymeleaf를 이용하여 정적인 순수HTML이 아닌 동적인 기능을 사용해보았습니다.

기능구현

  • 메인페이지 화면에서 홈, 넷플릭스, 왓챠, 웨이브, 모든 영화 보기, 자유게시판, 회원가입 버튼 / 검색창 기능 / 큰 이미지는 사진3개로 슬라이드 화면 구성

  • 넷플릭스 왓챠 웨이브 각 회사별 현제 상영중인 영화를 추천해주고 바로가기 버튼을 이용해서 각 회사의 홈페이지로 이동

  • 모든 영화 보기 모든 영화를 볼 수 있으며, 관리자는 영화등록 버튼으로 영화등록이 가능하며
    회원은 리뷰등록 버튼을 이용하여 리뷰와 평점등록이 가능

  • 자유게시판 영화추천 및 문의글 등록기능, 제목 내용 작성자 검색기능

  • 회원가입, 로그인 회원가입시 회원, 관리자로 나뉘어 권한부여

개선사항

  • 프로젝트 주제 선정 및 기획서 작성 10.12배웠던 기술들을 사용하여 자유게시판 하나의 기능은 구현하는데 성공하였으나 리뷰기능과 합쳐보니 에러코드가 났고, 에러를 해결하기위해 자유게시판과 리뷰기능의 소스코드와 경로를 다시 설정하여 문제를 해결하였습니다.

  • 회원가입기능에서 회원가입이 제대로 이루어 지지않아 repoitorytest에 회원과 관리자의 이메일을 임의의 값으로 데이터베이스에 저장시켜 회원과 관리자를 사용하였습니다.

  • 더 공부하여 평점을 데이터베이스에 저장시켜 각 영화의 제목옆에 평점칸을 만들어 자동으로 최신화시키는 작업을 추가하겠습니다.
