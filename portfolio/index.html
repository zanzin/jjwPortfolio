전진우 introduction

안녕하세요! 신입 개발자를 꿈꾸는 전진우입니다.

• 개발자를 꿈꾸기 전부터 도전을 즐거워하며 살다가, 지금은 개발의 매력에 푹 빠져 도전하고 있습니다.

• 사람들과의 많은 만남으로 나와 상대방의 다름을 이해하고 수용할 수 있는 마음과 존중하는 태도를 가지고 있습니다.

• 도전을 두려워하지 않으며, 항상 긍정적으로 웃으며 살아가고 있습니다.

• 가벼운 산책, 게임 그리고 영화 감상을 좋아합니다.

프로젝트 소개

인기있는 영화 드라마 정보를 한번에!

Netflix   Watcha   Wavve

요즘같이 코로나로 인해 집에서 여가시간을 보내는 사람들은 넷플릭스 같은 사이트에서 영화나 드라마를 자주 보곤 합니다. 그럴 때 보통 영화 추천이라고 검색을 하면 블로거나 유튜버 한 명의 의견이 많이 들어가있기 때문에 본인과 맞지 않은 영화를 시청하는 경우가 다분합니다.

위 사이트는 넷플릭스 왓챠 웨이브의 인기 있는 영상을 모두 볼 수 있게 추천하고, 많은 사람들의 리뷰로 보고 싶은 영화나 드라마를 빠르게 고를 수 있어 손쉽게 영상을 찾을 수 있습니다.

체크리스트 · DB 테이블 상세표

체크리스트 작성을 통해 필요 기능에 어떤 기능을 구현할지 생각해 보았고, 기능 설명을 통해 각 기능별 세부 기능들을 나눠서 계획을 짜보았습니다.
프로젝트를 진행하면서 리뷰 등록과 자유게시판에 문제점이 생겨 다시 체크리스트를 확인하여 방향을 잡을 수 있었습니다.

자유게시판과 리뷰의 번호에는 AUTO_INCREMENT 값으로 번호를 1씩 증가시켰습니다.

개발 환경


  • Framework
    Spring Boot :
    일일이 모든 설정을 하지 않아도 자주 사용되는 기본 설정을 알아서 해주어 많은 도움이 되었습니다.

  • language
    HTML · CSS · JavaSpript · SQL · Java :
    HTML로 전체적인 웹의 구조를 짜고 CSS로 정적인 디자인을 완성하고 JS로 동적인 기능들을 추가하였습니다.
    Java는 첫 프로젝트 진행시 사용하였으며, 변수 선언, 반복문, 클래스, 상속등 기본기를 익혀 자판기 기능을 구현해 보았습니다.

  • Tool
    Intellij · VisualStudio · HeidiSql · MySql · eclipse :
    데이터베이스로 MySql, HeidiSql, MariaDB를 이용하였으며, 프로젝트 진행시 벡 엔드, 프론트엔드에 다양한 도움을 주었고, 자동완성, 소스코드 분석 등 이클립스와 비교했을 때 너무 좋아서 잘 사용하였습니다.

  • Library
    jQuery · Thymeleaf :
    jQuery를 통해 직접 웹프론트에서 구동하도록 구현해 보았습니다.
    Thymeleaf를 이용하여 정적인 순수 HTML이 아닌 동적인 기능을 사용해 보았습니다.

  • shape management
    Github :
    GitHub를 이용하여 프로젝트의 소스 코드를 관리하고 있으며, GitHub 서버를 활용하여 개인 포트폴리오 사이트를 관리하고 있습니다.

기능구현

  • 메인페이지 화면에서 홈, 넷플릭스, 왓챠, 웨이브, 모든 영화 보기, 자유게시판, 회원가입 버튼 / 검색창 기능
    큰 이미지는 사진 3개로 슬라이드 화면 구성, bootstrap의 carousel으로 이미지 자동 전환 기능을 이용했습니다.

  • 메인페이지 body 넷플릭스 왓챠 웨이브의 인기 있는 영화, 드라마 3편의 정보 제공

  • 메인페이지 footer Top 버튼으로 이용해 페이지 가장 위쪽으로 이동

  • Netflix의 인기 있는 영화 드라마를 1페이지에 9개씩 구성하여 정보를 제공하며, 바로 가기 버튼을 이용해서 각 회사의 홈페이지로 이동

  • Watcha의 인기 있는 영화 드라마를 1페이지에 9개씩 구성하여 정보를 제공하며, 바로 가기 버튼을 이용해서 각 회사의 홈페이지로 이동

  • Wavve의 인기 있는 영화 드라마를 1페이지에 9개씩 구성하여 정보를 제공하며, 바로 가기 버튼을 이용해서 각 회사의 홈페이지로 이동

  • 모든 영화 보기 등록되어 있는 모든 영화·드라마 이미지를 볼 수 있으며, 번호 클릭 시 리뷰 등록 페이지로 넘어간다.

  • 영화 등록 관리자 로그인 시 이용 가능한 기능으로 영화 제목, 제공처, 이미지를 등록할 수 있다.

  • 영화 리뷰 번호를 클릭하여 들어온 페이지로 영화에 대한 자세한 정보(제목, 제공처, 리뷰 개수, 리뷰 평균, 등록 날짜, 수정 날짜, 이미지)를 볼 수 있다.
    리뷰 등록은 회원만 가능하며 버튼 클릭 시 평점과 리뷰 내용을 작성할 수 있다.

  • 자유게시판 영화 추천 및 문의 글 등록 기능, 제목 내용 작성자 검색 기능, 번호 클릭 시 게시판 수정 기능

  • 자유게시판 글쓰기 회원만 가능하며, 추천 영화 혹은 문의 글과 같은 자유로운 글 등록, 비로그인 시 로그인 창으로 넘어간다.

  • 자유게시판 글수정 번호를 클릭하여 들어온 페이지로 회원(user1@movie.com) 님 어서 오세요. 와 함께 수정 페이지로 넘어간다.
    수정페이지에서 내용만 수정가능하며 나머지는 readonly값으로 읽기기능만 가능하다.

  • 자유게시판 검색기능 찾고자 하는 제목 내용 작성자 별로 검색 기능

  • 회원가입, 로그인 회원가입 시 회원이 사용할 정보를 입력하고 회원, 관리자로 나뉘어 권한부여 가능

개선사항 및 느낀점

  • 배웠던 기술들을 사용하여 자유게시판 하나의 기능을 구현하는데 성공하였으나 리뷰 기능과 합쳐보니 에러코드가 났고, 에러를 해결하기위해 자유게시판과 리뷰 기능의 소스코드와 경로를 다시 설정하여 문제를 해결하였습니다.

  • 회원가입 기능에서 회원가입이 제대로 이루어지지않아 repoitorytest에 회원과 관리자의 이메일을 임의의 값으로 데이터베이스에 저장시켜 회원과 관리자를 사용하였습니다. 이 점은 더 보완하여 회원가입 기능을 구현할 수 있도록 하겠습니다.

  • 추가하고 싶지만 아직 구현하지 못한 작업
    평점을 데이터베이스에 저장시켜 각 영화의 제목 옆에 평점 칸을 만들어 자동으로 최신화시키는 작업을 추가하겠습니다.

java를 이용한 자판기 구현
(왼쪽 : 관리자, 오른쪽 : 사용자)

  • 평소에 자판에나 키오스크에 관심이 많아 java 개인 프로젝트를 진행시 자판기 기능을 구현해 보았습니다.

  • 저의 첫 프로젝트이자 관심 있는 자판기를 구현하는 것이라 많은 기능을 넣고 싶어 시간도 많이 걸렸고, 부족한 부분이 많아 구글에 검색도 많이 했습니다. 오류를 해결하면서 실력이 더 늘어나는 계기가 되었습니다.
